## Aim: 
To design a low-fidelity wireframe prototype for a mobile application using Balsamiq that includes essential screens such as Splash/Welcome, Login/Signup, Home/Dashboard, Product/Content Listing, and Checkout/Final Action, and to visualize the user flow between these screens in a consistent mobile layout.
## Algorithm:
**Step 1:** Open Balsamiq and create a new project.

**Step 2:** Insert a mobile phone frame with fixed dimensions.

**Step 3:** Design the Welcome/Splash screen with logo and start button.

**Step 4:** Create the Login/Signup screen with username and password fields.

**Step 5:** Design the Home/Dashboard screen with navigation and main content.

**Step 6:** Create the Product/Content Listing screen with item cards.

**Step 7:** Design the Checkout/Final Action screen with order summary and action button.

**Step 8:** Ensure all screens have the same mobile dimensions.

**Step 9:** Connect the screens using arrows to show user navigation flow.

**Step 10:** Save and export the completed wireframe.
